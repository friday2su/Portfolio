A premium, interactive portfolio website built for a Full Stack Developer & Web Designer. This project showcases a modern, dark-themed aesthetic with immersive 3D elements, smooth animations, and a focus on user experience.

![Portfolio Preview](https://i.ibb.co/dw368Gjy/image.png)

## ‚ú® Key Features

*   **Modern Aesthetic:** Dark theme with vibrant **Red** accents and glassmorphism effects.
*   **Interactive 3D Elements:** Integrated React Three Fiber scenes, including a rotating wireframe cube.
*   **Smooth Animations:** Powered by Framer Motion for page transitions and scroll reveals.
*   **Custom UI:**
    *   Unique "Terminal" style About section.
    *   Custom animated cursor with hover effects.
    *   Interactive click particle explosions.
*   **Responsive Design:** Fully optimized for all devices, from mobile to desktop.
*   **Project Showcase:** Dynamic grid layout to display GitHub projects and case studies.

## üöÄ Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/friday2su/Portfolio.git
    cd Portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
